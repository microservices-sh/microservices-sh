# Recurring Documents Admin Reference UI

Use this folder for admin route examples, table/list screens, forms, and action dialogs.

Expected permission: `recurring-documents.read` for read views and `recurring-documents.write` for mutations.

Expected screens:

- Recurring template list with document type, status, next run date, and generated count.
- Invoice/bill template form with schedule, max occurrences, end date, tax, account defaults, and line items.
- Actions for pause, resume, cancel, generate now, and run due cycle.
