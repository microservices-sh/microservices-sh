# Membership Credits Admin Reference UI

Expected admin surfaces:

- Membership tier list and edit forms.
- Customer active membership assignment/change/cancel actions.
- Customer credit balance and transaction ledger.
- Credit grant/debit forms with required descriptions and approval gates.

Expected permission: `membership-credits.read` for read views, `membership-credits.membership-admin` for tier/membership changes, and `membership-credits.credit-admin` for balance changes.
