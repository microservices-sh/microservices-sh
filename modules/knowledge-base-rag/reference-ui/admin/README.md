# Knowledge Base RAG Admin Reference UI

Use this folder for admin route examples, table/list screens, forms, and action dialogs.

Expected permission: `knowledge-base-rag.read` for read views and `knowledge-base-rag.write` for mutations.

Expected admin screens:

- Articles list with status, source type, indexed state, and updated time.
- Article editor with title, content, source URL, and archive/restore actions.
- Source and attachment panels that show ingestion status without storing raw files directly.
- Support answer tester that displays citations and refusal states.
