# Estimate Quote Admin Reference UI

Use this folder for admin route examples, table/list screens, forms, and action dialogs.

Expected permission: `estimate-quote.read` for read views and `estimate-quote.write` for mutations.

Expected screens:

- Quote table with status filters and pending/accepted/converted value summaries.
- Draft quote form with client, expiry, notes, terms, tax basis points, discount cents, and line items.
- Detail actions for send, mark viewed, accept, decline, void, and convert to invoice.
