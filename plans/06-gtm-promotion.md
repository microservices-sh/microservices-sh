# GTM And Promotion Plan

## GTM Strategy
Start narrow:

> AI-heavy developers and agencies building production apps with Claude, Codex, or Cursor.

Do not market to everyone building software. Market to people already feeling agent-generated app pain.

## Core Message
Primary:

> Build production apps with your AI agent, using verified Cloudflare-native modules.

Supporting:

> Compose Auth, Customer, Booking, Order, Payment, Email, and Admin. Customize safely. Deploy to managed Cloudflare without setup.

Pricing/value frame:

> Cloudflare gives you primitives. microservices.sh gives your agent verified modules, safe customization, tests, deploys, and upgrades.

## MCP Directory Layer
Use MCP directories as an early high-intent discovery channel, not as the whole launch.

Submit after the create command, MCP server, and setup docs are credible:

1. Official MCP Registry
2. Glama MCP Directory
3. PulseMCP
4. MCP.so

Directory positioning:

> Verified app modules for AI coding agents.

Measure directory traffic by:

- MCP setup page visits
- CLI quickstart visits
- create command attempts
- successful MCP connections
- first `list_templates` calls
- first `compose_app` calls
- discovery calls booked from directory traffic

See `13-mcp-directory-distribution.md`.
