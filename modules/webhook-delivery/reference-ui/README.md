# Webhook Delivery Reference UI

Admin: endpoint registration, delivery attempts, signing-secret status, retry diagnostics, and test delivery controls.

ERP shell example: `templates/erp-shell-sveltekit/src/routes/app/webhooks`.

Visitor: not applicable. Webhook delivery is external integration infrastructure.

Agentic: inspect endpoint and delivery history first. Require approval before registering endpoints or triggering deliveries.
