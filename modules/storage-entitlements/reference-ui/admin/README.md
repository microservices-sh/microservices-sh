# Storage Entitlements Admin Reference UI

Use this folder for admin route examples, table/list screens, forms, and action dialogs.

Expected permission: `storage-entitlements.read` for read views and `storage-entitlements.write` for mutations.

Expected screens:

- Storage usage panel with quota, used bytes, remaining bytes, and usage percentage.
- Storage package table with price, storage bytes, active/popular flags, and provider ids.
- Share-link list with short id, expiry, download count, and revoke action.
